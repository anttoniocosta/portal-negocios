# Fluxos do Sistema

## Fluxo do Comprador
1. Acessa portal
2. Filtra ativos
3. Visualiza detalhes
4. Interage com consultor
5. Realiza proposta

## Fluxo do Vendedor
1. Realiza pré-cadastro
2. Envia ativo
3. Aguarda aprovação
4. Ativo publicado

## Fluxo do Consultor
1. Recebe ativo
2. Estrutura informações
3. Submete para aprovação
4. Atende interessados

## Fluxo Administrativo
1. Analisa ativo
2. Aprova ou rejeita
3. Publica
