# portal-negocios
Plataforma digital para negociação de ativos de alto valor, como imóveis, empresas, fazendas e veículos, permitindo compra, venda ou locação em um único ambiente. Funciona como infraestrutura de mercado, com governança, histórico e rastreabilidade, diferindo de imobiliárias, marketplaces comuns ou redes sociais, com potencial de evolução futura
